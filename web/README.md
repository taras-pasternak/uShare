# uShare Web

Фронтенд застосунку `uShare`, який надає прогресивний веб-досвід для зберігання та обміну соціальними профілями.

## Старт
- Встановіть залежності: `npm install`
- Запустіть локально: `npm run dev`

## Стан
> Поки що це лише каркас. Детальні інструкції зʼявляться після початку розробки.
